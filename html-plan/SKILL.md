In a `.plans` directory (create if not already there) in the root of the codebase, create a thorough, well-researched plan for an implemenation that is aesthetically pleasing to a human and has visualization for humans to understand if necessary. Use [PRODUCT.md](PRODUCT.md) and [DESIGN.md](DESGIN.md) for the design you should use for this. The plan should have a timestamp of when it was created.
